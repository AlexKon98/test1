Необходимо на JavaScript реализовать выравнивание высоты строк в адаптивном макете.
Исходная разметка (запрещено менять html и/или добавлять стили) https://jsfiddle.net/x2tbhdon/2/


Начальные условия:
— Есть адаптивным макет с карточками товаров. Ширина карточки фиксированная для каждого разрешения. Высота зависит от контента.
— На разрешении lg и md присутствует проблема: карточки перепрыгивают в предыдущую строку, образуются "дыры".
- Необходимо реализовать "выравнивание строк" на JavaScript. Т.е. на каждом разрешении (в том числе при изменении размеров браузера) высота каждой строки должна ровняться высоте максимального элемента в ней (именно в строке, а не среди всех карточек)
- Можно использовать: vanilla js (ES5/6/7), jQuery, React/vue
- Нельзя использовать: готовые плагины

Не используйте пожалуйста решения из поисковой выдачи, stackoverflow и т.д. - мы все равно поймем это, просто немного позже.
Напишите решение самостоятельно.
